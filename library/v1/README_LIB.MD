#### 엔티티 상태
##### LIBRARY
- `강한 엔티티` : 
  - 독립적으로 존재
  - 엔티티 내에서 자체 식별자를 사용하여 고유하게 인스턴스 식별 가능
- `독립 엔티티` : 사람, 물건 처럼 원래 현실에 존재하는 엔티티
- `핵심 엔티티` : 부모 엔티티를 갖지 않는 독립적이며 핵심적인 엔티티
##### BOOK
- `강한 엔티티` :

    - 독립적으로 존재

    - 엔티티 내에서 자체 식별자를 사용하여 고유하게 인스턴스 식별 가능

- `독립 엔티티` : 사람, 물건 처럼 원래 현실에 존재하는 엔티티

- `핵심 엔티티` : 부모 엔티티를 갖지 않는 독립적이며 핵심적인 엔티티


#### 관계
- 책과 도서관은 서로 독립적인 관계
